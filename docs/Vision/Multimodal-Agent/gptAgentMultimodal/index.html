    • 核心组件

      • 感知组件关注处理多模态信息
      • 规划器负责推理和制定计划
      • 行动组件执行计划
      • 记忆组件则涉及长期和短期记忆

    • 四种类型

      • 无长期记忆的闭源 LLMs 作为规划器
      • 无长期记忆的微调 LLMs 作为规划器
      • 具有间接长期记忆的规划器
