    textually prompted models #

    • contrastive CLIP 双塔
    • generative Flamingo
    • hybrid BLIP
    • conversational GPT-4， miniGPT4, LLaVa

    传统上，视觉语言模型主要用于需要同时理解视觉和文本模态的任务。然而，随着CLIP展示出的卓越性能，基于语言监督的模型在显著上升，并成为主流方法。在本节中，我们专注于探索依赖语言作为主要监督来源的方法。这些以文本为提示的模型可以广泛分为三种主要类型：对比、生成和混合方法。
